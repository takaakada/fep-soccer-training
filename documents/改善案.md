
# ベイズの定理で考える

## 「プレイAが成功したとき、動作Bができていたと言えるか？」

あるスポーツにおいて、**動作Bが重要**だと言われているとします。

たとえば、次のように考えます。

* 動作Bがうまくできているとき、プレイAが成功する確率は **60%**
* 動作Bがうまくできていないとき、プレイAが成功する確率は **30%**
* その選手は、そもそも動作Bをうまくできる割合が **20%** だとする

ここで知りたいのは、

> **プレイAが成功したとき、そのとき本当に動作Bができていた確率はどのくらいか？**

ということです。

直感的には、

> プレイAが成功したのだから、動作Bもできていたのでは？

と考えたくなります。
しかし、ベイズの定理で考えると、必ずしもそうとは限りません。

---

## 1. 前提を確率で表す

記号を次のように置きます。

* (AT)：プレイAが成功した
* (AF)：プレイAが失敗した
* (BT)：動作Bがうまくできた
* (BF)：動作Bがうまくできなかった

前提は以下です。

$P(AT \mid BT)=0.6$
$P(AT \mid BF)=0.3$
$P(BT)=0.2$
$P(BF)=0.8$

---

## 2. 1万回やったとするとどうなるか

1万回プレイAをしたと仮定します。

まず、動作Bができる割合は20%なので、

* 動作Bができる：2000回
* 動作Bができない：8000回

です。

そのうえで、

* 動作Bができるときは60%成功
* 動作Bができないときは30%成功

なので、表はこうなります。

|             | 動作Bできる (BT) | 動作Bできない (BF) |    合計 |
| ----------- | ----------: | -----------: | ----: |
| プレイA成功 (AT) |        1200 |         2400 |  3600 |
| プレイA失敗 (AF) |         800 |         5600 |  6400 |
| 合計          |        2000 |         8000 | 10000 |

---

## 3. 成功したとき、Bができていた確率は？

プレイAが成功したのは合計3600回です。
そのうち、動作Bができていたのは1200回です。

したがって、


$P(BT \mid AT)=\frac{1200}{3600}=\frac{1}{3}\approx 0.33$

つまり、

> **プレイAが成功したとしても、そのとき動作Bができていた確率は33%**

です。

---

## 4. ベイズの定理で書くと

ベイズの定理より、


$P(BT \mid AT)=\frac{P(AT \mid BT)P(BT)}{P(AT)}$

ここでまず全体の成功率 (P(AT)) を求めます。

$P(AT)=P(AT \mid BT)P(BT)+P(AT \mid BF)P(BF)$
$P(AT)=0.6\times0.2+0.3\times0.8$
$P(AT)=0.12+0.24=0.36$

したがって、

$P(BT \mid AT)=\frac{0.6\times0.2}{0.36}
=\frac{0.12}{0.36}
=\frac{1}{3}$

やはり **33%** になります。

---

## 5. 何が言えるのか

ここで重要なのは、

* 動作Bができると成功しやすい
* しかし、動作Bができなくても30%は成功する
* しかも、そもそも動作Bができる割合が20%と低い

という点です。

そのため、

> **「成功した」という事実だけでは、「そのときBができていた」とはまだ言い切れない**

のです。

つまり、

> **Bは重要かもしれないが、成功1回だけを見てBができたと判断するのは危険**

というのがベイズ的な見方です。

---

# では、プレイAの成功率を50%に上げたいとしたら？

次に考えたいのは、

> プレイAの成功率を、今の36%ではなく **50%** に上げたい
> そのためには、動作Bができる割合はどのくらい必要か？

という問いです。

前提は変えずに、

* $(P(AT \mid BT)=0.6)$
* $(P(AT \mid BF)=0.3)$

のままとします。

知りたいのは $(P(BT))$ です。

---

## 6. 全体成功率50%になる条件

全体成功率は


$P(AT)=P(AT \mid BT)P(BT)+P(AT \mid BF)P(BF)$

なので、

$0.5=0.6P(BT)+0.3(1-P(BT))$
$0.5=0.6P(BT)+0.3-0.3P(BT)$
$0.5=0.3P(BT)+0.3$
$0.2=0.3P(BT)$
$P(BT)=\frac{2}{3}\approx 0.667$

つまり、

> **プレイAの成功率を50%にしたいなら、そもそも動作Bをうまくできる割合を66.7%まで上げる必要がある**

ということになります。

---

## 7. このときの1万回表

1万回プレイAをしたとすると、

* 動作Bができる：6667回
* 動作Bができない：3333回

と考えられます。

すると、

|             | 動作Bできる (BT) | 動作Bできない (BF) |    合計 |
| ----------- | ----------: | -----------: | ----: |
| プレイA成功 (AT) |       約4000 |        約1000 |  5000 |
| プレイA失敗 (AF) |       約2667 |        約2333 |  5000 |
| 合計          |       約6667 |        約3333 | 10000 |

---

## 8. 成功したときに、Bができていた確率は？

このとき、

$P(BT \mid AT)=\frac{4000}{5000}=0.8$

つまり、

> **成功したときに、動作Bができていた確率は80%**

です。

ベイズの定理でも同じです。

$P(BT \mid AT)=\frac{P(AT \mid BT)P(BT)}{P(AT)}
=\frac{0.6\times0.667}{0.5}
\approx 0.8$

---

# まとめ

この話からわかることは次の通りです。

### ① 成功したからといって、Bができていたとは限らない

最初の設定では、

* 動作Bができる割合：20%
* Bができると成功率60%
* Bができなくても成功率30%

なので、成功してもそのときBができていた確率は **33%** にとどまります。

---

### ② 成功の背景を見るには、事前確率が重要

ベイズの定理では、

* Bがどれくらいできる人なのか
* Bができない状態でもどのくらい成功してしまうのか

が効いてきます。

つまり、

> **成功だけではなく、その成功がどんな母集団から出てきたかが大切**

です。

---

### ③ 成功率を上げたいなら、「成功時の形」だけでなく「普段からBができる割合」を上げる必要がある

プレイAの成功率を50%まで上げたいなら、

> **動作Bができる割合を66.7%まで高める必要がある**

という計算になります。

---

  <script>
    const ids = ["p_bt", "p_at_bt", "p_at_bf", "target"];
    function syncPair(baseId) {
      const range = document.getElementById(baseId);
      const num = document.getElementById(baseId + "_num");
      range.addEventListener("input", () => { num.value = range.value; update(); });
      num.addEventListener("input", () => {
        let v = Number(num.value);
        if (Number.isNaN(v)) v = 0;
        v = Math.max(Number(range.min), Math.min(Number(range.max), v));
        range.value = v;
        num.value = v;
        update();
      });
    }
    ids.forEach(syncPair);

    const fmtPct = (x) => `${(x * 100).toFixed(1)}%`;
    const fmtN = (x) => Math.round(x).toLocaleString("ja-JP");

    function update() {
      const pBT = Number(document.getElementById("p_bt").value) / 100;
      const pAT_BT = Number(document.getElementById("p_at_bt").value) / 100;
      const pAT_BF = Number(document.getElementById("p_at_bf").value) / 100;
      const target = Number(document.getElementById("target").value) / 100;
      const pBF = 1 - pBT;

      const pAT = pAT_BT * pBT + pAT_BF * pBF;
      const pAF = 1 - pAT;
      const pBT_AT = pAT > 0 ? (pAT_BT * pBT) / pAT : 0;
      const pBT_AF = pAF > 0 ? ((1 - pAT_BT) * pBT) / pAF : 0;

      const N = 10000;
      const atBt = N * pBT * pAT_BT;
      const afBt = N * pBT * (1 - pAT_BT);
      const atBf = N * pBF * pAT_BF;
      const afBf = N * pBF * (1 - pAT_BF);

      document.getElementById("out_pat").textContent = fmtPct(pAT);
      document.getElementById("out_pbt_at").textContent = fmtPct(pBT_AT);
      document.getElementById("out_pbt_af").textContent = fmtPct(pBT_AF);

      document.getElementById("cell_at_bt").textContent = fmtN(atBt);
      document.getElementById("cell_at_bf").textContent = fmtN(atBf);
      document.getElementById("cell_at_total").textContent = fmtN(atBt + atBf);
      document.getElementById("cell_af_bt").textContent = fmtN(afBt);
      document.getElementById("cell_af_bf").textContent = fmtN(afBf);
      document.getElementById("cell_af_total").textContent = fmtN(afBt + afBf);
      document.getElementById("cell_bt_total").textContent = fmtN(N * pBT);
      document.getElementById("cell_bf_total").textContent = fmtN(N * pBF);

      const formula = [
        `P(AT) = P(AT|BT)×P(BT) + P(AT|BF)×P(BF)`,
        `      = ${pAT_BT.toFixed(2)}×${pBT.toFixed(2)} + ${pAT_BF.toFixed(2)}×${pBF.toFixed(2)}`,
        `      = ${pAT.toFixed(4)}  (${fmtPct(pAT)})`,
        ``,
        `P(BT|AT) = [P(AT|BT)×P(BT)] / P(AT)`,
        `         = [${pAT_BT.toFixed(2)}×${pBT.toFixed(2)}] / ${pAT.toFixed(4) || "0"}`,
        `         = ${pAT > 0 ? pBT_AT.toFixed(4) : "0.0000"}  (${fmtPct(pBT_AT)})`
      ].join("\n");
      document.getElementById("formula_box").textContent = formula;

      const denom = pAT_BT - pAT_BF;
      const reqBox = document.getElementById("out_req");
      const msgBox = document.getElementById("message_box");

      if (Math.abs(denom) < 1e-9) {
        if (Math.abs(target - pAT_BT) < 1e-9) {
          reqBox.textContent = "任意";
          reqBox.className = "metric-value accent";
          msgBox.textContent = "Bができてもできなくても成功率が同じ設定です。この場合、目標成功率がその値と同じなら P(BT) は何%でも構いません。";
        } else {
          reqBox.textContent = "不可能";
          reqBox.className = "metric-value bad";
          msgBox.textContent = "Bの有無で成功率が変わらない設定です。このため、P(BT) を増やしても全体成功率は変わりません。";
        }
        return;
      }

      const requiredPBT = (target - pAT_BF) / denom;
      if (requiredPBT < 0 || requiredPBT > 1) {
        reqBox.textContent = "不可能";
        reqBox.className = "metric-value bad";
        const minP = Math.min(pAT_BT, pAT_BF);
        const maxP = Math.max(pAT_BT, pAT_BF);
        msgBox.textContent = `この設定では全体成功率は ${fmtPct(minP)} 〜 ${fmtPct(maxP)} の範囲にしかなりません。目標の ${fmtPct(target)} はこの範囲外なので、P(BT) だけを動かしても到達できません。`; 
      } else {
        reqBox.textContent = fmtPct(requiredPBT);
        reqBox.className = "metric-value good";
        msgBox.textContent = `目標の ${fmtPct(target)} を達成するには、動作Bの再現率 P(BT) を ${fmtPct(requiredPBT)} まで上げる必要があります。現在の設定は ${fmtPct(pBT)} です。`;
      }
    }
    update();
